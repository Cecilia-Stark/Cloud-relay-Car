# 端口使用与占用说明

本文档详细列出了 5G 远程驾驶云平台中所有涉及的端口、对应的服务以及潜在的冲突说明。

## 1. 核心服务端口 (云服务器 119.45.14.204)

| 端口 | 所属服务 | 协议 | 用途说明 |
| :--- | :--- | :--- | :--- |
| **8080** | Web 前端 | HTTP | PM2 托管的 Nginx/Vite 预览服务 (`ecosystem.config.cjs`)。 |
| **8082** | G29 中继 | WebSocket | 广播 G29 方向盘实时数据 (`g29-relay-server.cjs`)。 |
| **8083** | G29 中继 | HTTP | 接收 Python 脚本发送的 G29 数据 (`g29-relay-server.cjs`)。 |
| **8088** | SRS 媒体 | HTTP-FLV | 提供 HTTP-FLV 视频流拉流服务 (`srs-*.conf`)。 |
| **9001** | 认证服务 | HTTP | 用户登录、Token 签发、录制会话管理 (`replay-server.js`)。 |
| **13306**| 数据库 | TCP | MySQL 数据库服务端口 (`.env` 配置)。 |

## 2. 流媒体与 SRS 端口 (云服务器)

| 端口 | 所属服务 | 协议 | 用途说明 |
| :--- | :--- | :--- | :--- |
| **1935** | SRS 媒体 | RTMP | 接收摄像头/路端设备推流 (`srs-*.conf`)。 |
| **1985** | SRS 媒体 | HTTP API | SRS 内部信令接口，用于 WebRTC 握手 (`srs-*.conf`)。 |
| **8000** | SRS 媒体 | WebRTC | SRS WebRTC 推流与播放服务 (`srs-*.conf`)。 |

## 3. 开发与辅助端口 (本地/局域网)

| 端口 | 所属服务 | 协议 | 用途说明 |
| :--- | :--- | :--- | :--- |
| **8001** | 视频代理 | HTTP | 代理小车视频流，解决跨域和鉴权 (`video-proxy-*.cjs`)。 |
| **6999** | G29 直连 | WebSocket | 本地读取 USB G29 硬件数据 (`g29-server.cjs`)。 |

## 4. 小车端设备端口 (192.168.0.5 或 4G 路段)

| 端口 | 所属服务 | 协议 | 用途说明 |
| :--- | :--- | :--- | :--- |
| **8000** | 控制接口 | HTTP/TCP | 接收控制指令 (`carControl.ts` 发送目标)。 |
| **8080** | ROS2 视频 | HTTP | 运行 `web_video_server` 提供原始图像流 (`video-proxy-ultra.cjs` 目标)。 |
| **6999** | 小车控制 | TCP | 接收 G29 Python 脚本发送的底层控制指令 (`send-g29-to-car-and-server.py`)。 |

---

## ⚠️ 端口冲突与注意事项

虽然以下端口号相同，但由于运行在不同的物理机器上，目前**没有产生冲突**：

1.  **端口 8000 (WebRTC vs 控制接口)**：
    *   **服务器**使用 `8000` 提供 SRS WebRTC 服务。
    *   **小车**使用 `8000` 提供控制接口。
    *   *建议*：若将来通过 SSH 隧道将小车接口映射到服务器，必须更换映射目标（如 `18000`），否则会冲突。

2.  **端口 8080 (Web前端 vs 小车视频)**：
    *   **服务器**使用 `8080` 运行 Web 前端页面。
    *   **小车**使用 `8080` 运行 ROS2 视频流。
    *   *现状*：物理隔离，互不影响。

## 5. 数据流路径概览

```text
[物理设备] 
   |--> 小车 8000 (控制) <--- [浏览器] POST /control
   |--> 小车 8080 (原始图像) --- [代理:8001] --> [浏览器]
   |--> 摄像头 1935 (RTMP推流) --> [SRS:1935] --> [SRS:8088 HTTP-FLV] --> [浏览器]
   |--> G29 脚本 8083 (HTTP录入) --> [服务器:8083] --> [广播:8082 WS] --> [浏览器]
```
